# AI 예산편성도우미 사용설명서

**대상:** 예산편성 심사정보시스템 사용자(소관부서 예산 담당·심사 담당 직원)  
**버전:** 2026년 6월  
**문의:** 예산담당관실(시스템·운영 담당)

---

## 1. AI 예산편성도우미란?

예산편성 심사정보시스템에 **2013년부터 축적된 경상·투자 심사조서**를 대상으로, 정해진 규칙에 따라 질문하면 **심사조서에서 검색·정리된 결과**를 돌려주는 보조 도구입니다.

소관부서별로 입력·보고한 사업별 **요구·반영·조정반영·검토의견**이 담긴 심사조서를, 과거 연도·타 부서 사례 참고나 유사 사업 검색 시 활용할 수 있습니다.

---

## 2. 일반 AI 챗봇과 다른 점

| 구분 | 일반 AI 챗봇(ChatGPT 등) | AI 예산편성도우미 |
|------|---------------------------|-------------------|
| **데이터 출처** | 인터넷 학습 데이터·임의 생성 | **심사정보시스템 DB(심사조서)만** 조회 |
| **답변 방식** | 자유 형식·추측 가능 | **정해진 보고 서식**으로만 정리 |
| **연결 환경** | 외부 인터넷 API | **부산시 내부 행정 AI**(내부망 전용) |
| **데이터 변경** | 해당 없음 | **조회만 가능**(입력·수정·삭제 불가) |
| **검색 대상** | 제한 없음 | **경상사업심사조서·투자사업심사조서** 중심 |
| **금액 표기** | 임의 단위 가능 | **백만원** 단위, **조정액(반영액)** 기준 재원 표시 |

**핵심:** 이 도우미는 “아무 질문에 대답하는 만능 AI”가 아니라, **시스템에 등록된 심사조서를 규칙에 맞게 찾아 정해진 양식으로 보여 주는 검색·정리 도구**입니다. DB에 없는 사업·내용은 찾을 수 없으며, 답변에 없는 수치를 임의로 만들지 않습니다.

## 3. 시스템에서의 이용 방법

### 3.1 화면 위치

- 예산편성 심사정보시스템 **로그인 후 모든 주요 화면 하단 중앙**에 **「AI 예산편성 도우미」** 패널이 표시됩니다.
- 패널 제목을 누르면 펼치거나 접을 수 있습니다.

### 3.2 질문 보내기

1. 챗 입력창에 질문을 입력합니다.
2. **Enter** → 전송 / **Shift+Enter** → 줄바꿈
3. 「전송」 버튼을 눌러도 됩니다.
4. 잠시 후 챗 창에 **답변 텍스트**와 필요 시 **검색 결과 표**가 표시됩니다.

### 3.3 활용 시점(예)

- 올해 편성·심사 중 **과거 동일·유사 사업**의 조정·검토 사례 확인
- 특정 **키워드·시행기관·검토 표현**이 들어간 사업 목록 조회
- **소관 실국·부서** 범위로 사업을 좁혀 검색
- 여러 연도(예: 2018~2022년)에 걸친 사업 이력 참고

## 4. 질문 입력 규칙(포맷·팁)

질문은 **아래 요소를 조합**하면 정확도가 높아집니다. 자연어로 물어도 되지만, **대괄호 표기·연도·조서 구분**을 넣으면 더 잘 찾습니다.

### 4.1 기본 질문 구조

```
[회계연도] + [조서 구분] + [검색 조건] + [찾아달라는 말]
```

**예시**

```
2026년 경상사업 및 투자사업에서 유가보조금 사업을 찾아서 정리해줘
```

### 4.2 회계연도

| 표기 | 설명 |
|------|------|
| `2026년` | 해당 연도 심사조서 검색 |
| `2018년~2022년` | 연도 범위 검색 |
| 연도 생략 | 최신 연도·인근 연도 순으로 검색 시도 |

### 4.3 조서 구분(경상 / 투자)

| 질문에 포함 | 검색 범위 |
|-------------|-----------|
| `경상사업` (투자 없음) | 경상사업심사조서만 |
| `투자사업` (경상 없음) | 투자사업심사조서만 |
| `경상사업 및 투자사업` 또는 둘 다 언급 | **양쪽 조서** 모두 |

### 4.4 사업명·키워드 검색(가장 많이 사용)

- 사업명·세세사업명·통계목·검색태그 등에 들어 있는 **핵심 단어**를 넣습니다.
- `사업`이라는 말 없이 `유가보조금 찾아줘`처럼 물어도 됩니다.
- 여러 키워드는 쉼표로 구분 가능합니다.  
  예: `유가보조금, 유류비 관련 사업`
- **영문**은 대·소문자 구분 없이 검색됩니다. (`IT` = `it`)

**예시**

```
2025년 경상사업 및 투자사업에서 일상돌봄 사업을 찾아줘
```

### 4.5 [소관부서] — 부서·실국으로 범위 한정

부서·실국 소속 사업만 보고 싶을 때 **`[소관부서]` 대괄호를 반드시** 붙입니다.

```
2026년 [소관부서]복지국 유가보조금 사업을 찾아서 정리해줘
```

| 표기 | 동작 |
|------|------|
| `[소관부서]복지국` + 사업명 | **사업명 검색** + 복지국(실국·부서) 소속만 |
| `[소관부서]복지국` 만 | 해당 부서·실국 소속 사업 전체(사업명 없을 때) |
| `복지국` (대괄호 없음) | 부서 필터 **적용 안 됨** — 일반 키워드 검색 |

### 4.6 [구분] · [검토내용] — 비정형(요구·검토) 검색

심사조서 화면의 **[구분]**(요구내용)·**[검토내용]** 필드 안의 문장을 찾을 때 사용합니다.  
**반드시 `[구분]` 또는 `[검토내용]`을 대괄호로 적어** 주세요.

**[구분] 검색 예시** (시행주관·시행처·기관명 등)

```
2026년 경상사업 및 투자사업에서 [구분]에 있는 내용 중 테크노파크가 시행처인 사업을 찾아서 정리해줘
```

**[검토내용] 검색 예시**

```
2026년 경상사업 및 투자사업에서 [검토내용]에 있는 내용 중 마무리 사업을 찾아서 정리해줘
```

**[소관부서]와 함께**

```
2026년 [소관부서]복지국 [검토내용] 마무리 사업을 찾아서 정리해줘
```

### 4.7 시행주관·시행처 관련 질문

`시행주관`, `시행주체`, `시행처`, `시행기관`, `사업기관` 등의 표현이 있으면 **[구분](요구내용)** 에서 기관명을 검색합니다.

```
2026년 경상사업에서 테크노파크가 시행주관인 사업을 찾아줘
```

### 4.8 차수 구분(선택)

필요 시 질문에 포함합니다.

- `본예산`
- `1회추경`, `2회추경` 등

### 4.9 잘 묻는 방법 · 피할 표현

**권장**

- 연도 + (경상/투자) + 찾을 키워드를 한 문장에 명확히
- 부서 한정 시 `[소관부서]부서명`
- 요구·검토 문구 검색 시 `[구분]` 또는 `[검토내용]`
- “찾아줘”, “정리해줘”, “검색해줘” 등으로 마무리

**비권장**

- 연도·사업 구분 없이 “예산 알려줘”만 입력
- 부서명만 쓰고 `[소관부서]` 생략(부서 필터 미적용)
- 심사조서에 없는 내용을 사실처럼 요구(“왜 반영 안 했는지 이유 말해줘” 등 — DB에 없으면 답 불가)
- 개인정보·대외비 문서 내용 입력

## 5. 결과값이 나오는 방식

### 5.1 기본 답변 서식(사업 1건당)

별도로 요청하지 않으면 **아래 4항목만** 표시됩니다.

| 순서 | 항목 | 내용 |
|------|------|------|
| 1 | **[사업명(통계목)]** | 세세사업명 + 통계목 코드. 예) `지역사랑상품권 인센티브 보상금 ( 308-13)` |
| 2 | **[소관부서]** | 실국·부서명 |
| 3 | **[차수별 예산내역(재원표시)]** | 차수별 **조정액(반영액)** 총액(백만원) + 괄호 안 **재원별 조정액**(국비·시비 등) |
| 4 | **[차수별 요구, 검토의견]** | 차수별 요구·검토 핵심 요약 |

**금액 예시**

```
본예산: 200백만원(국비140, 시비60)
```

- 표시 단위: **백만원**
- 괄호 안 재원: **조정액 기준** 재원 구성(요구액 재원이 아님)
- 같은 사업에 본예산·추경이 여러 건이면 **한 사업 블록 안에서 차수별로** 구분

### 5.2 명시적으로 요청할 때만 추가되는 항목

질문에 아래를 **직접** 넣은 경우에만 3번 [차수별 예산내역]에 추가 표시됩니다.

| 요청 표현 | 추가 표시 |
|-----------|-----------|
| `요구액` | 차수별 요구액 |
| `전년도예산`, `전년`, `기정액` | 전년도 예산액 |
| `시행주관`, `시행처` 등 | [구분] 관련 내용 |

**예시**

```
2026년 ○○사업의 요구액과 반영액, 전년도 예산액을 알려줘
```

### 5.3 검색 결과 표

- 일부 질문에서는 답변 아래 **표 형태**로 조회 행이 함께 나올 수 있습니다.
- 표 하단에 `총 N건 (표시 행수 제한 적용)` 안내가 있을 수 있습니다.

### 5.4 결과가 없을 때

- “조건에 맞는 심사조서 데이터가 없습니다” 등의 안내와 함께 **예시 질문**이 제시될 수 있습니다.
- 연도·키워드·대괄호 표기를 바꿔 다시 질문해 보세요.

## 6. 주의사항

### 6.1 보안·운영

- **내부망·행정망**에서만 사용합니다. 질문 내용은 내부 AI 서버로 전달됩니다.
- **주민번호, 연락처, 대외비** 등 불필요한 개인·민감 정보는 입력하지 마세요.
- 챗봇 답변은 **업무 참고용**이며, **공식 심사·결재 문서를 대체하지 않습니다.** 최종 판단은 심사조서 원본·담당자 확인을 기준으로 합니다.
- 시스템 데이터는 **조회만** 하며, 챗봇으로 예산·조서를 **수정·삭제할 수 없습니다.**

### 6.2 검색·답변의 한계

| 항목 | 한계 |
|------|------|
| **데이터 범위** | 시스템에 등록·마감된 심사조서만 검색(2013년~ 최신 연도) |
| **건수 제한** | 한 번에 조회·표시되는 사업 수에 **상한(약 50건)** 이 있어, 많이 걸리면 일부만 표시될 수 있음 |
| **미등록 사업** | 아직 조서에 없거나 다른 명칭으로만 등록된 사업은 검색 안 될 수 있음 |
| **해석** | 검토의견은 **요약** 형태이며, 법적·회계적 최종 해석이 아님 |
| **표시 제외** | 기본 답변에 **조건검색어(#태그), 총사업비, 조서구분** 등은 넣지 않음(별도 요청 시 일부 가능) |
| **행정운영경비** | 정책사업 중 기본경비·인력운영비 등 **심사 대상에서 제외된 사업**은 검색에서 빠질 수 있음 |

### 6.3 질문이 잘 안 될 때 점검

1. **회계연도**를 넣었는가?
2. **경상 / 투자** 구분을 넣었는가?
3. 부서 한정이 필요하면 **`[소관부서]`** 를 썼는가?
4. 요구·검토 문구 검색이면 **`[구분]` / `[검토내용]`** 을 썼는가?
5. 키워드를 **짧고 구체적으로**(사업명·기관명 핵심어) 썼는가?

## 8. 용어 정리

| 용어 | 설명 |
|------|------|
| **심사조서** | 경상사업심사조서·투자사업심사조서 화면에 보고하는 예산 심사 결과 문서 |
| **요구액** | 부서가 요구한 예산액 |
| **조정액(반영액)** | 심사 후 반영·조정된 예산액 — 기본 답변의 “예산액”은 이 금액 기준 |
| **[구분]** | 조서의 요구내용·시행주체 등이 적힌 항목 |
| **[검토내용]** | 조서의 검토의견·심사 의견이 적힌 항목 |
| **조건검색어** | `#민생` 등 조서에 붙인 태그(기본 답변에는 미표시) |
